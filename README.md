# ObsiMap Beta

**베타 테스터 전용** 저장소입니다. 커뮤니티 플러그인 스토어용이 아닙니다.

정식(공개) 버전: [Creative781/obsimap](https://github.com/Creative781/obsimap)

이 저장소에는 설치에 필요한 빌드 파일만 포함됩니다. 소스 코드는 비공개로 유지됩니다.

---

## 설치

### BRAT (권장)

1. [BRAT](https://github.com/TfTHacker/obsidian42-brat) 플러그인을 설치합니다.
2. **Settings → Community plugins → BRAT → Add Beta plugin**
3. 저장소 URL: `https://github.com/Creative781/obsimap_dev`
4. Obsidian을 다시 로드한 뒤 **ObsiMap**을 활성화합니다.

### 수동 설치

1. [Releases](https://github.com/Creative781/obsimap_dev/releases)에서 최신 버전의 `main.js`, `manifest.json`, `styles.css`를 받습니다.
2. 보관함 `.obsidian/plugins/obsimap/` 폴더에 넣습니다. (폴더 이름은 `obsimap`입니다.)
3. 설정에서 플러그인을 켭니다.

## ObsiMap Beta (English)

**Beta testers only.** Not for the Obsidian Community Plugins directory.

Public release: [Creative781/obsimap](https://github.com/Creative781/obsimap)

This repository ships built plugin files only (no source).

### Install with BRAT

1. Install [BRAT](https://github.com/TfTHacker/obsidian42-brat).
2. **Settings → Community plugins → BRAT → Add Beta plugin**
3. Repository: `https://github.com/Creative781/obsimap_dev`
4. Reload Obsidian and enable **ObsiMap**.

### Manual install

Download `main.js`, `manifest.json`, and `styles.css` from [Releases](https://github.com/Creative781/obsimap_dev/releases) into `.obsidian/plugins/obsimap/`.
